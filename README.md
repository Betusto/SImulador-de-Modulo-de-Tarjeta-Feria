# SImulador-de-Modulo-de-Tarjeta-Feria
Programa de escritorio que consiste en simular un módulo de tarjeta feria en horario pico, usando distribuciones para desarrollar un generador de números aleatorios. 2019.<br>
Lenguaje usado: Java.
